# Psycle Class Finder — Development Setup

## Prerequisites

- macOS
- Python 3 (comes with Xcode Command Line Tools)
- Node.js (v20+)
- Xcode (for iOS builds)
- CocoaPods (`sudo gem install cocoapods` if not installed)

## Web App (PWA)

```bash
# Clone and serve
git clone <your-repo-url> psycle-booking
cd psycle-booking
python3 -m http.server 8080
```

Open http://localhost:8080/psycle-finder.html

That's it — no build step, no npm install for the web app.

## iOS App (Capacitor)

The `ios/` directory is NOT committed — it's generated by Capacitor. You must bootstrap it on each new machine.

### First time setup (fresh clone)

```bash
# From the repo root
cd ios-app

# Install Node dependencies (Capacitor + plugins)
npm install

# Generate the iOS Xcode project (creates ios/ directory)
npx cap add ios

# Sync web assets into the iOS project
# (copies js/, css/, HTML from root into ios-app/www/, runs pod install)
npm run sync

# Open in Xcode (opens the .xcworkspace, not .xcodeproj)
npx cap open ios
```

### If the ios/ directory exists but is broken

```bash
cd ios-app
rm -rf ios
npx cap add ios
npm run sync
npx cap open ios
```

### Node version issues

If `npx` or `node` isn't found, ensure your Node version is set:
```bash
# If using asdf
asdf install nodejs 20.19.5
asdf local nodejs 20.19.5

# If using nvm
nvm install 20
nvm use 20
```

### Xcode setup (first time only)

1. After `npx cap open ios`, Xcode opens the **workspace**
2. **IMPORTANT**: Always open `App.xcworkspace`, never `App.xcodeproj`
3. Select your development team: **Signing & Capabilities** → **Team**
4. Set the bundle identifier: `com.psyclefinder.app` (or your own)
5. Select a connected iPhone or simulator
6. **Product → Run** (or `Cmd+R`)

### Rebuilding after code changes

```bash
# After editing any js/ or css/ files:
cd ios-app
npm run sync

# Then in Xcode: Cmd+R to rebuild
```

Or use the shortcut:
```bash
cd ios-app && npm run build
# (runs sync + opens Xcode)
```

### Common issues

| Problem | Fix |
|---------|-----|
| `Unable to open base configuration reference file .xcconfig` | Run `cd ios-app/ios/App && pod install` or nuke + re-add ios |
| `capacitor.config.json: No such file or directory` | Run `npx cap add ios` — the ios/ directory wasn't generated |
| `public: No such file or directory` | Same as above — `npx cap add ios` creates the public/ folder |
| `pod: command not found` | `sudo gem install cocoapods` |
| `No version is set for command node` | Set node version via asdf/nvm (see above) |

## App Icon & Assets in Xcode

### App Icon

1. In Xcode, open the project navigator (left sidebar)
2. Navigate to **App → App → Assets.xcassets → AppIcon**
3. You'll see a grid of icon slots at different sizes
4. The icon files are in `ios-app/appstore-assets/`:

| File | Xcode slot |
|------|-----------|
| `AppIcon-20.png` | Notification 20pt @1x |
| `AppIcon-40.png` | Notification 20pt @2x, Spotlight 40pt @1x |
| `AppIcon-58.png` | Settings 29pt @2x |
| `AppIcon-60.png` | Spotlight 20pt @3x |
| `AppIcon-76.png` | iPad App 76pt @1x |
| `AppIcon-80.png` | Spotlight 40pt @2x |
| `AppIcon-87.png` | Settings 29pt @3x |
| `AppIcon-120.png` | iPhone App 60pt @2x, Spotlight 40pt @3x |
| `AppIcon-152.png` | iPad App 76pt @2x |
| `AppIcon-167.png` | iPad Pro App 83.5pt @2x |
| `AppIcon-180.png` | iPhone App 60pt @3x |
| `AppIcon-1024.png` | App Store 1024pt @1x |

**Easiest method**: Drag `AppIcon-1024.png` into the **App Store 1024pt** slot. Xcode 15+ can auto-generate all sizes from the single 1024px icon. To enable this:
- Select **AppIcon** in Assets.xcassets
- In the Attributes Inspector (right sidebar), set **iOS** sizes to **"Single Size (iOS 17+)"**
- Drag `AppIcon-1024.png` into the single slot

**Manual method**: Drag each PNG into its matching slot one by one.

### Launch Screen

Capacitor generates a default white launch screen. To customize:
1. In Xcode: **App → App → Assets.xcassets → Splash**
2. Replace the splash image, or
3. Edit `ios/App/App/Base.lproj/LaunchScreen.storyboard` directly

### App Display Name

To change the name shown under the icon on the home screen:
1. In Xcode: select the **App** target → **General** tab
2. Change **Display Name** to `Psycle`

### Status Bar Style

Already configured in `capacitor.config.json`:
```json
{
  "ios": {
    "preferredContentMode": "mobile",
    "backgroundColor": "#0a0a0a"
  }
}
```

The native bridge (`www/native-bridge.js`) auto-sets the status bar style based on dark/light theme.

### App Store Screenshots

Upload the framed screenshots from `ios-app/appstore-assets/`:
1. Log in to [App Store Connect](https://appstoreconnect.apple.com)
2. Go to your app → **App Information** → **Screenshots**
3. Upload the `01-` through `06-` framed PNGs (1290x2796, 6.7" iPhone)
4. Upload `AppIcon-1024.png` as the App Store icon

Full App Store copy (description, keywords, etc.) is in `ios-app/APP_STORE_LISTING.md`.

## Playwright (for automated screenshots / testing)

```bash
pip3 install playwright
python3 -m playwright install chromium
```

Then from the repo root:
```bash
python3 -m http.server 8080 &
python3 -c "
from playwright.sync_api import sync_playwright
with sync_playwright() as p:
    browser = p.chromium.launch(headless=False)
    page = browser.new_page()
    page.goto('http://localhost:8080/psycle-finder.html')
    input('Press Enter to close...')
    browser.close()
"
```
